# String Manipulation Algorithms

This repository contains solutions to various string manipulation algorithms, written in Java.

## Solutions

* Reverse String
* Count Vowels
* Reverse Words
* Are Rotations
* Remove Duplicates
* Most Repeated Character
* Capitalize Words
* Is Anagram
* Is Palindrome
* Longest Palindrome Substring
* Has Balanced Brackets
